# github-cheat-sheet
Lage en readme file
```sh
echo "# github-cheat-sheet" >> README.md
```

Lage local repo
```sh
git init
```
Legg filen til commit
```
git add README.md
```
lage commit
```sh
git commit
```
Lage en branch og gi den navnet main
```sh
git branch -M main
```

link local repo til github
```sh
git remote add "name" "url" 
```
Pushe filene fra local til github
```sh
git push -u origin main
```
